# YourDadJokes: Joke Extraction Interface

**YourDadJokes** is a single page web application that functions as a random joke generator. The project itself is inspired by 1970's analog sci-fi hardware. The app allows users to fetch randomized jokes from the icanhazdadjoke.com API. 
The interface is terminal styled with high contrast typography. 

---

## Why Does It Exist?
This project is an unserious way for users to find entertainment quickly while taking a break from daily tasks. It is also a low distraction application so its users do not get too far away from their tasks in the office or school. 

---

## What Tools Did I Use?
- **VS Code** - I used VS Code for my text editor, with the Live Server extension to view changes made to my code base in real time.
- **Bumla CSS** - Bulma was chosen because it was lightweight, had a flexbox grid system, so I could easily override the styling to support more of a retro aesthetic.
- **Google Fonts** - I used Google Fonts, specifically Barlow & JetBrains Mono to capture the "Black/900" industrial Atari styled weight for headers and terminal output.
- **GitHub Pages** - GitHub pages was used for its free hosting, quick deployment, and reliability.
- **Gemini AI** - I used Gemini to help me with dealing with fetching errors, and polishing the bold typography CSS.
- **Web Speech API** - This tool is a browser based feature to support text to speech accessibility.
